# plans/superseded/

Implementation Plan Documents (IPDs) that were REPLACED by a better or subsequent plan -
kept for the record, not the live path.

To retire a plan here, prepend a header
`RETIRED YYYY-MM-DD: <reason>; superseded by <path/commit>` and `git mv` the file into this
directory (preserving history). **Never silently delete** a plan - the record of what we
tried and why we changed course is valuable. Use `not-executed/` instead if the plan was
dropped with no replacement.
