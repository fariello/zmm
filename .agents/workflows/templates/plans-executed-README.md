# plans/executed/

Implementation Plan Documents (IPDs) that were implemented, verified, and tested.

Terminal, append-only history: do not edit past records to match a later design. (`done/`
is an accepted alias for this directory if a repo already uses it.) A plan that was NEVER
run does not belong here - retire it to `superseded/` or `not-executed/` instead, so this
directory never falsely claims something was implemented.
