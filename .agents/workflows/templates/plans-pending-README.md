# plans/pending/

Implementation Plan Documents (IPDs) that are queued or under review/implementation.

Named `YYYYMMDD-HHMM-NN-<slug>.md`. Reviewed (optionally with `/plan-review`) and approved
by a human before execution. **Move to `executed/` ONLY after the change is implemented,
verified, and tested** (with a post-execution summary). If a plan is dropped instead of
run, retire it to `superseded/` or `not-executed/` - never leave an abandoned plan here.
