# plans/not-executed/

Implementation Plan Documents (IPDs) we DELIBERATELY decided not to run, with no
replacement - explored and rejected, or overtaken by events.

To retire a plan here, prepend a header
`RETIRED YYYY-MM-DD: <reason>` (and, if relevant, what it was overtaken by) and `git mv`
the file into this directory (preserving history). **Never file an un-run plan in
`executed/`** (that falsely claims implementation), and **never silently delete** it - the
record of a considered-and-rejected idea is valuable. Use `superseded/` instead if a
successor plan replaced it.
