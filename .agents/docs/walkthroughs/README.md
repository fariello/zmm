# .agents/docs/walkthroughs/

Narrative execution walkthroughs documenting the implementation, verification, and testing results of executed plans.

Named `YYYYMMDD-HHMM-NN-<slug>-walkthrough.md` (local time).

Every executed plan should have a corresponding walkthrough here. Walkthroughs capture command logs, test results, and screenshots or recording paths showing that the plan was fully implemented and validated.
